# Carpeta de Cartas Personalizadas

Coloca aquí las imágenes PNG de tus cartas generadas con el CardCreator.

## Estructura de carpetas:

```
cards/
├── players/      - Cartas de jugadores (ej: mama.png, papa.png, fay.png, etc)
├── wrappers/     - Cartas de envoltorios (ej: rojo.png, verde.png, azul.png, etc)
└── ribbons/      - Cartas de cintas (ej: roja.png, verde.png, azul.png, etc)
```

## Nombres de archivo requeridos:

### Jugadores (players/):
- mama.png
- papa.png
- fay.png
- fio.png
- tato.png
- raffa.png

### Regalos (wrappers/):
- regalo-1.png
- regalo-2.png
- regalo-3.png
- regalo-4.png
- regalo-5.png
- regalo-6.png

### Cintas (ribbons/):
- roja.png
- verde.png
- azul.png
- dorada.png
- plateada.png
- blanca.png

## Instrucciones:

1. Genera todas tus cartas en el CardCreator (/card-creator)
2. Descarga cada carta como PNG
3. Renombra los archivos según la lista de arriba (todo en minúsculas, sin tildes)
4. Coloca cada imagen en su carpeta correspondiente
5. Las imágenes aparecerán automáticamente en el juego

## Notas:

- Los nombres deben estar en minúsculas y sin espacios
- El formato debe ser PNG
- Tamaño recomendado: 750x1050px (generado automáticamente por CardCreator)
- Si falta alguna imagen, se mostrará un placeholder con ícono
